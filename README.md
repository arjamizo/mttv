# MTTV

MT-Silesia TV Project

Program allowing composing polaroid (thumbnails of multiple images) of all images in given folder into existing images

# How to run

    # git clone...
    mkdir img
    #for: wget random_image > ./img/IMG_$i
    ./mttv_wizard.sh
